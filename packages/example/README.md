# the/example
Example package used by Analyzer tests

## Installation

```bash
the install the/example
```

## Example usage

```the
import example from "the/example"
```

## API

### `example (text: str) str`
Returns text passed as parameter.

**Parameters**

- `text` - text to return

**Return value**

Text passed as parameter.

**Examples**

```the
example("greeting")
```
